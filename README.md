Небольшое веб-приложение для отображения списка товаров.

## Стек
- React
- JavaScript
- Vite
- CSS

## Реализовано
1)отображение списка товаров
2)поиск по названию
3)обработка небольших опечаток в поиске
4)фильтрация по категории
5)фильтрация по диапазону цены
6)сортировка товаров
7)модальное окно с подробной информацией о товаре
8)адаптивная верстка
9)переключение темы

## Особенности реализации
- логика поиска, фильтрации и сортировки вынесена в util-функции
- загрузка товаров и логика темы вынесены в кастомные хуки
- категории для фильтра формируются автоматически на основе данных
- для поиска используется сочетание прямого совпадения и расстояния Левенштейна

## Структура проекта
- `components/` — UI-компоненты
- `hooks/` — кастомные хуки
- `pages/` — страницы приложения
- `utils/` — логика поиска и фильтрации
- `public/data.json` — mock-данные товаров

## Запуск проекта

```bash
cd front
npm install
npm run dev

```

Примечание - данные(data.json) были сгенерированы с помощью ИИ в целях экономии времени.
